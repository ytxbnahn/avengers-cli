## 介绍
用于快速生成模版
### 模版

### 用法
```javascript
yarn global add avengers-cli
or
npm install -g avengers-cli
avergers init my-app
```

### 需要使用的第三方库
babel-cli/babel-env: 语法转换<br>
commander: 命令行工具<br>
download-git-repo: 用来下载远程模板<br>
ini: 格式转换<br>
inquirer: 交互式命令行工具<br>
ora: 显示loading动画<br>
chalk: 修改控制台输出内容样式<br>
log-symbols: 显示出 √ 或 × 等的图标<br>
